# YourCoverAI Mobile App

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Add platforms:
   ```bash
   npx cap add ios
   npx cap add android
   ```

3. Sync:
   ```bash
   npx cap sync
   ```

4. Open in IDE:
   ```bash
   npx cap open ios
   npx cap open android
   ```

## App Icon

Place your app icon (1024x1024 PNG) at: assets/app-icon.png

## Configuration

The app loads the web version from: https://yourcoverai.billyus2025.workers.dev

To change this, edit capacitor.config.json
